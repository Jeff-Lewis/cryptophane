Menu item functions

The following describes the functionality of each menu item available in the Cryptophane application. The top-level item indicates the main menu option and each list item is a submenu option.

File

  • Encrypt: opens a file browser to select a file to encrypt. Once a file is selected the Encryption window is displayed
  • Sign: opens a file browser to select a file to sign. Once a file is selected the Encryption window is displayed
  • Encrypt and Sign: opens a file browser to select a file to encrypt and sign. Once a file is selected the Encryption window is displayed
  • Decrypt: opens a file browser to select an encrypted file to decrypt. Once selected you will be promted to enter your passphrase
  • Verify Signature: opens a file browser to select a file to verify the signature of
  • Message: opens a new window which allows you to enter or paste text that will be encrypted. Once a file is selected the Encryption window is displayed. The resulting encrypted text will be presented in a window and also added to the clipboard for easy pasting
  • Import Keys: opens a file browser to select a key to import
  • Export Public keys: opens a window with a list of all the Public keys on your keyring. You can select one or more keys and export them to either a file or to the screen and clipboard
  • Export Secret keys: opens a window with a list of all your Secret keys on your keyring. You are warned before being able to select one or more keys and export them to either a file or to the screen and clipboard
  • Exit: closes Cryptophane

View

  • Folders: toggle the display of the Folders pane on the main screen
  • Expired keys: toggle the display of expired keys on the main screen
  • Hints: toggle the display of the Hints pane on the main screen
  • Public Keys / Secret Keys: toggle between displaying the public keys and your secret keys in the keys grid of the main screen. If the Secret keys option is selected, the folders pane is also hidden (secret keys cannot be assigned to folders)
  • Icons / Details: toggle between displaying a simple 'icon' view or a more verbose 'detail' view for each key in the keys grid of the main screen

Keys

  • Properties of Selected Key...: displays the Properties window for the selected key
  • Sign Selected Key...: Opens the Sign Key window for the selected key
  • Edit Trust of Selected Key...: Opens the Edit Level of Trust window for the selected key
  • Delete Selected Key: Opens a prompt window confirming whether you really want to delete the selected key. Selecting 'Yes' from this window will delete the key from your keyring. Selecting 'No' will close the prompt window and the selected key will not be deleted
  • Search Keyserver: Opens the Search Keyserver window to allow you to find and add new public keys to your keyring
  • Get Key from Keyserver: Opens the Get Key from Keyserver window. This is faster than a search if you already know the key ID you want to get
  • Refresh from Keyserver: Updates all the keys from the Keyserver
  • Send to Keyserver: Sends the selected key to the keyserver. Use this function on a key you have just signed
  • Generate secret key: Opens the Generate Secret Key window. Use this function to create a new secret key. You generally won't need to use this again until your key expires
  • Refresh: Updates the main screen view

Tools

  • Options: Opens the Crytophane Options window which allows you to configure program options and defaults

Help

  • Contents: Open this help file in the Contents view
  • Index: Open this help file in the Index view
  • Cryptophane Web Site...: Opens the Crytophane web site in your default web browser
  • About...: Opens the 'About Cryptophane' window which displays version, copyright and legal information about Cryptophane

Interface functions

Right mouse-click on a key

  • Key Properties: displays the Properties window for the selected key.
  • Encrypt to...: opens a file browser to select a file to encrypt. Once a file is selected the Encryption window is displayed with the selected key selected for encryption
  • Sign key...: Opens the Sign Key window for the selected key
  • Edit Trust...: Opens the Edit Level of Trust window for the selected key
  • Delete Key: Opens a prompt window confirming whether you really want to delete the selected key. Selecting 'Yes' from this window will delete the key from your keyring. Selecting 'No' will close the prompt window and the selected key will not be deleted
  • Send to Keyserver...: Sends the selected key to the keyserver. Use this function on a key you have just signed
  • Refresh from default Keyserver...: Updates the selected key from the default Keyserver

Enter text into the 'Filter' input field

Entering text in the filter input field will perform a real-time filtering of keys in the key grid. Keys the do not match the filter input will not be displayed in the key grid. Clearing the text from the filter input field will display all the keys in the key grid.
