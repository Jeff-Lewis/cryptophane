What is Cryptophane?

A User's Perspective

Cryptophane is an application that lets you:

  • Encrypt files for sending so no-one but your intended receipients can read them
  • Sign files so that others can verify that you originally sent the file
  • Decrypt files that were encrypted using Cryptophane, PGP, GnuPG, or any other OpenPGP-compliant application
  • Verify signatures that were generated using Cryptophane, PGP, GnuPG, or any other OpenPGP-compliant application

In addition to the four basic cryptographic functions, it allows you to:

  • Import and export keys from your Cryptophane keyring
  • Generate new secret keys
  • Search, receive and send keys from/to keyservers on the Internet
  • Browse your keyring and view each key's properties
  • Sign keys on your keyring
  • Edit your trust of the keys on your keyring

To learn the basics of cryptography, view the Cryptography Primer. Use the Contents menu to the left to find out more about Cryptophane's features.

A Technical Perspective

Cryptophane "wraps" the GnuPG application to provide a visual, easy-to-use frontend. Instead of intimidating users new to cryptography with a command-line interface, Cryptophane allows all commonly used cryptographic functions to be accessed via a familiar Windows interface.
